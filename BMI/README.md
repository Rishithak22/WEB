# 🧮 BMI Calculator Web Application

A simple and responsive **BMI (Body Mass Index) Calculator** built using **HTML, CSS, Bootstrap, and JavaScript**.
This web application allows users to calculate their BMI by entering their **height and weight**, and it instantly displays the BMI value along with the health category.

---

## 🚀 Live Demo

🔗 **View the Project:**
https://rishithak22.github.io/WEB/BMI/index.html

---

## 📌 Features

* 📏 User inputs for **Height and Weight**
* 🧮 Calculates **BMI instantly**
* 📊 Displays **BMI value and health category**
* 🔄 **Reset button** to clear inputs
* 🎨 Clean and responsive **Bootstrap design**

---

## 🧠 BMI Formula

BMI is calculated using the formula:

BMI = Weight (kg) / Height² (m²)

---

## 📊 BMI Categories

| BMI Range    | Category      |
| ------------ | ------------- |
| Below 18.5   | Underweight   |
| 18.5 – 24.9  | Normal Weight |
| 25 – 29.9    | Overweight    |
| 30 and above | Obese         |
